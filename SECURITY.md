# Security Policy

## Reporting a vulnerability

**Please do not file a public issue for security vulnerabilities.** Public reports expose the issue to attackers before a fix is available.

Email vulnerability reports to **<security-contact@example.com>** (replace with a real address before publishing). Include a description, reproduction steps, and impact assessment.

You should receive an acknowledgment within 7 days and an initial assessment within 14 days.

## Supported versions

Only the latest released version receives security fixes unless explicitly noted in the release notes.
